# Портал вакансий

Веб-приложение для подачи и администрирования заявок на вакансии. Реализовано на PHP фреймворке Laravel.

## Технологии
- **PHP 8.2**
- **Laravel 11**
- **MySQL** (База данных)
- **Blade & Tailwind CSS** (Фронтенд)
- **MVC Architecture**

## Функционал
1. **Форма заявки:** Валидация данных, защита от спама (CSRF).
2. **Админ-панель:**
   - Просмотр списка всех кандидатов.
   - Скрытый вход для администраторов.
   - Защита маршрутов через Session Auth (пароль).
   - Кнопка безопасного выхода (Logout).
3. **Безопасность:**
   - Защита от SQL-инъекций (ORM Eloquent).
   - Защита от XSS (Blade escaping).

## Как запустить (Installation)

1. Клонировать репозиторий:
git clone https://https://github.com/ViktorVox/LaravelSite.git

2. Установить зависимости:
composer install

3. Создать файл окружения:
cp .env.example .env

4. Настроить базу данных в файле `.env` и запустить миграции:
php artisan migrate

5. Запустить сервер:
php artisan serve
